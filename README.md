# Advanced Weather App 🌤️

A simple weather application built using **HTML**, **CSS**, and **JavaScript**, fetching real-time weather data from the **OpenWeatherMap API**.  

It allows users to search for any city and see current weather conditions in a clean, responsive interface.

---

## Features

- 🔹 **City Search** – Enter a city name to get weather updates.  
- 🔹 **Current Weather** – Shows temperature, humidity, wind speed, and conditions.  
- 🔹 **Temperature Unit Toggle** – Switch between °C and °F (optional).  
- 🔹 **Dynamic Background & Icons** – Background changes based on weather type.  
- 🔹 **Responsive Design** – Works on both mobile and desktop devices.
